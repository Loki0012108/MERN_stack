Experience & Projects

Project: codetraxora

CodeTraxora is a offline code Generator tool that helps users generate and upload IoT code for ESP8266 and ESP32 boards. It allows users to enter Wi-Fi and Blynk details, map virtual pins to GPIO pins, and automatically create an Arduino sketch. The application compiles and uploads the code to selected boards using Arduino CLI. Its user-friendly interface simplifies IoT development for beginners and students. The project is useful for smart home automation and rapid IoT prototyping.
